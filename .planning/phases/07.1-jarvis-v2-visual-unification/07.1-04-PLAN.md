  <action>
Create `dither-shader.js` — a WebGL-based halftone ditherpunk shader that processes media presenter images in real-time, inspired by bogdanrosu.net/halftone/.

1. **DitherShader class**:
   - Creates an offscreen WebGL canvas (same dimensions as media presenter image area).
   - Fragment shader implements circular halftone dot pattern:
     ```glsl
     // For each pixel, calculate distance to nearest dot center in a grid
     // Dot size proportional to luminance of original image at that point
     // Dark areas = large dots, bright areas = small/no dots
     // Apply slight rotation per CMYK channel to avoid moire patterns
     ```
   - Vertex shader: simple fullscreen quad.
   - Uniforms: `u_texture` (source image), `u_dotSize` (4.0-12.0 pixels), `u_dotSpacing` (8.0-16.0), `u_angle` (rotation in radians), `u_colorMode` (0=monochrome gold, 1=CMYK, 2=deity-tinted).
   - `processImage(imgElement)`: Takes an HTMLImageElement, uploads to WebGL texture, renders through shader, returns the result as a canvas that can be composited over the original.

2. **Integration approach**: Instead of replacing the img element, overlay the WebGL canvas on top of the media presenter image when dithering is active. The WebGL canvas sits inside the media-presenter div with `position: absolute; inset: 0; mix-blend-mode: multiply;` (or similar compositing mode that achieves the halftone look).

3. **Real-time updates**: When the media presenter changes slides (media_slide event), re-process the new image through the shader. Use `requestAnimationFrame` for smooth animation of the halftone pattern — the dot grid can slowly rotate (0.01 rad/frame) for a living, breathing effect.

4. **Update dither-toggle.js**: Replace the SVG filter approach with the WebGL shader:
   - When `toggle()` is called with dithering ON: create/show the WebGL overlay canvas, process current media image.
   - When OFF: hide/remove the WebGL overlay.
   - Keep the 'd' keyboard shortcut and HUD button.
   - The SVG filter code can remain as fallback for browsers without WebGL2 support.

5. **Deity color tinting**: When `u_colorMode` is 2 (deity-tinted), use the current deity palette to tint the halftone dots. Gold (#daa520) dots for zeus/2501, teal (#00cc88) for greek, etc. Read the current voice from `window.gradientBackground?._activeVoice || 'default'`.

6. **Performance**: Only render the shader when dithering is active AND media presenter is visible. Do NOT run the WebGL pipeline every frame when nothing is showing.

Add `<script src="/static/js/dither-shader.js"></script>` before dither-toggle.js in index.html.

Create `s9-logos-text.js` — a terminal-aesthetic rotating 3D text effect inspired by bogdanrosu.net/s9logos/. This is a UI accent element, not a full-screen takeover.

1. **S9LogosText class**:
   - Creates a fixed-position container at z-index 5 (behind blob, subtle background accent). Position: bottom-left corner, 200x120px area. `pointer-events: none; opacity: 0.15` (very subtle).
   - Generates SVG text elements containing monospace characters: Greek letters (alpha, beta, gamma, delta, omega), deity names in uppercase, circuit symbols, occult/alchemical Unicode glyphs.
   - Characters arranged in a 3D grid (using CSS perspective transform on parent, individual character translateZ values for depth).
   - Slow Y-axis rotation (CSS animation, 20-30 second period) creates the 3D carousel effect.
   - Characters cycle through content every 5-10 seconds with a brief glitch effect (random character scramble for 200ms before settling on new text).

2. **Glitch effect**: On text change, rapidly cycle through random Unicode characters for 150-200ms before settling. Use `requestAnimationFrame` with a frame counter — swap characters every 2 frames for 10 frames, then show final text.

3. **Voice reactivity**: When voice changes, update the character set to include the active deity's name and associated symbols. During speech, increase opacity to 0.25 and speed up rotation slightly.

4. **CSS 3D setup**:

   .s9-logos-container {
     perspective: 400px;
     transform-style: preserve-3d;
   }
   .s9-logos-ring {
