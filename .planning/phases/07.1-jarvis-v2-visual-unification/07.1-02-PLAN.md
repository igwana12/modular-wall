1. **NebulaParticleSystem class**:
   - Creates a full-viewport canvas at z-index 12 (above blob iframe at z:10, below deity canvas at z:15, below panels). `pointer-events: none`.
   - Particle pool: 200-400 particles. Each particle has: position (x,y), velocity, size (2-8px), opacity (0.1-0.5), color (from deity palette), lifetime.
   - Particles orbit loosely around screen center (where blob is) in an elliptical cloud, radius 150-350px from center. Use simplex noise or perlin for organic drift — NOT circular orbits.
   - Particle rendering: Use canvas 2D with `globalCompositeOperation: 'screen'` for additive blending. Draw each particle as a radial gradient circle (soft glow, not hard dot).

2. **Voice reactivity**:
   - Read `window.audioEngine.ttsEnergy` each frame.
   - During speech (energy > 0.05): increase particle velocity by 2x, expand orbit radius by 30%, increase opacity to 0.3-0.6, spawn 10-20 extra particles per second.
   - During silence: particles drift slowly, fade to base opacity (0.1-0.2), orbit contracts.
   - On onset (energy spike > 0.7): spawn a burst of 30 particles outward (shockwave ring effect).

3. **Deity color palette**: Reuse the DEITY_GRADIENT colors from gradient-background.js. When voice changes:
   - `setVoice(voiceKey)`: Lerp all particle colors toward the new deity palette over 2 seconds. Use the `from` and `mid` colors from DEITY_GRADIENT, but at higher opacity/saturation than the background gradient (particles should be visible, not washed out).
   - Default palette (no deity): deep purple + teal + gold sparks.

4. **Oracle mode**: Add `setOracleMode(active)` that activates during oracle readings:
   - Shifts to sacred purple palette (`#7c3aed`, `#daa520`, `#00cc88`)

   - Electric flash effect: occasionally (every 3-8 seconds) a bright particle streak crosses the cloud (the "laser" effect the user described)

5. **Performance**: Use requestAnimationFrame but skip frames if FPS drops below 30. Cap particle count at 400. Reuse particle objects from a pool instead of creating/destroying.

6. **Integration in index.html**: Add script tag and initialize: `window.nebulaParticles = new NebulaParticleSystem()`. Wire `setVoice()` alongside existing voice-change handlers. Wire `setOracleMode()` to oracle_step WebSocket events.
