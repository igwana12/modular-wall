```json
"context_profiles": {
  "_comment": "Directory-based context profiles. Advisory — recommends --add-dir directories for sessions based on task type. Detection uses working directory (D-13).",

      "services/jarvis"
    ],

  },
  "jarvis": {
    "description": "JARVIS voice AI frontend and media engine",
    "detect_dirs": ["services/jarvis"],
    "include": [
      "services/jarvis",
      ".planning"
    ],
    "exclude": [
      "firmware",
      "oracle-app"
    ],
    "tags": ["frontend", "voice", "media", "visualization"]
  },
  "hardware": {
    "description": "ESP32 firmware and hardware documentation",
    "detect_dirs": ["firmware"],
    "include": [
      "firmware",
      ".planning"
    ],
    "exclude": [
      "services/jarvis",
      "oracle-app"
    ],
    "tags": ["esp32", "arduino", "led", "audio", "iot"]
  },
  "creative": {
    "description": "Sacred Circuits content, art assets, and mythology corpus",

    ],
    "exclude": [
      "firmware",
      "services"
    ],
    "tags": ["content", "art", "mythology", "writing"]
  }
}
```

Each profile has:
- `description`: Human-readable purpose
- `detect_dirs`: Directories that trigger this profile when they are the working directory (per D-13)
- `include`: Directories to add to session context
- `exclude`: Directories to exclude from session context
- `tags`: Keywords for task classification matching

Preserve ALL existing sections exactly as they are: `_comment`, `recurring_tasks`, `interactive_tasks`, `monthly_budget_estimate`. Do not modify any existing key or value.
