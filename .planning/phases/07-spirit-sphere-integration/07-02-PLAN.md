<interfaces>
From scripts/image-to-pov.py (Phase 6):
- CLI tool: python scripts/image-to-pov.py input.png --output header.h --mode sphere --cols 120 --rows 36 --gamma 2.2 --brightness 255
- Outputs: C header with const uint8_t PROGMEM FRAME_DATA[] and #define FRAME_DATA_LEN
- Mode "sphere" = equirectangular projection (maps rectangular image to sphere surface)
- Mode "flat" = direct pixel mapping for propeller display

From firmware/pov-globe/frame_buffer.h:
- void frame_load(const uint8_t* data, size_t len);
- Expects BGR byte order (3 bytes per LED * NUM_LEDS * NUM_COLUMNS)
- Frame size: 36 LEDs * 120 columns * 3 bytes = 12,960 bytes
</interfaces>

**hardware/wiring-diagram.md** -- Pin-to-Pin Connection Guide:

ASCII art diagram + detailed connection table showing every wire from ESP32-S3-BOX-3 to peripherals.

Key connections:
1. **ESP32-S3-BOX-3 expansion header -> Slip ring (stationary side)**:
   - GPIO 11 (SPI MOSI) -> 100 ohm -> Slip ring wire 3 (DATA)
   - GPIO 12 (SPI SCK) -> 100 ohm -> Slip ring wire 4 (CLK)
   - 5V rail -> Slip ring wire 1 (VCC)
   - GND -> Slip ring wire 2 (GND)
   - Slip ring wires 5-6: spare

2. **Slip ring (rotating side) -> APA102 LED strip**:
   - Slip ring wire 1 (VCC) -> APA102 5V
   - Slip ring wire 2 (GND) -> APA102 GND
   - Slip ring wire 3 (DATA) -> APA102 DI (data in)
   - Slip ring wire 4 (CLK) -> APA102 CI (clock in)

3. **ESP32-S3-BOX-3 -> Motor**:
   - GPIO 5 -> N-channel MOSFET gate (or direct if motor draws <40mA)
   - MOSFET drain -> Motor terminal 1
   - Motor terminal 2 -> 5V rail
   - Flyback diode across motor terminals

4. **ESP32-S3-BOX-3 -> Hall sensor**:
   - GPIO 4 -> US5881LUA output (with 10K pullup to 3.3V)
   - 3.3V -> US5881LUA VCC
   - GND -> US5881LUA GND

5. **ESP32-S3-BOX-3 -> Mute button + LED**:
   - GPIO 39 -> Momentary button -> GND (INPUT_PULLUP)
   - GPIO 40 -> 220 ohm -> Red LED anode; LED cathode -> GND

6. **Battery system**:
   - 3x 18650 in 3S holder -> 3S BMS board
   - BMS board USB-C port -> external charger
   - BMS output -> LM2596 buck converter input
   - LM2596 output (5V) -> ESP32-S3-BOX-3 USB-C power (or 5V pin on expansion header)
   - LM2596 output (5V) -> APA102 VCC (via slip ring)

Include a caution note: "NEVER connect 3S battery (9-12.6V) directly to ESP32 or LEDs. Always go through buck converter."

Include a note about SPI_SPEED being set to 6MHz (not 12MHz) for slip ring reliability per RESEARCH.md pitfall 4.
