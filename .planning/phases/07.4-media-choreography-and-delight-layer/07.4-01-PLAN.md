In index.html, replace the single `.viz-panel` transition rule with a comprehensive CSS transition library. Add these CSS keyframes and classes inside the existing `<style>` block:

**7 named transition keyframes (enter + exit pairs):**
1. `fade` — opacity 0->1 (enter), 1->0 (exit). The existing behavior, refined with better easing.
2. `scale` — scale(0.85) + opacity 0 -> scale(1) + opacity 1. Panel grows into view.
3. `reveal` — clipPath inset(100% 0 0 0) -> inset(0). Curtain reveal from bottom.
4. `slide-up` — translateY(40px) + opacity 0 -> translateY(0) + opacity 1. Slides up into position.
5. `wipe` — clipPath inset(0 100% 0 0) -> inset(0). Left-to-right wipe.
6. `split` — scaleX(0) + opacity 0 -> scaleX(1) + opacity 1. Horizontal split open.
7. `zoom` — scale(1.3) + opacity 0 -> scale(1) + opacity 1. Zoom in from larger.

**Easing curves:**
- Enter: `cubic-bezier(0.16, 1, 0.3, 1)` (smooth deceleration)
- Exit: `cubic-bezier(0.7, 0, 0.84, 0)` (smooth acceleration)

**Base `.viz-panel` update:**
- Remove `transition: opacity 0.6s ease` — transitions now applied per-class
- Keep `opacity: 0; pointer-events: none; display: none;`
- Add `.viz-panel.visible` keeps `opacity: 1; pointer-events: auto; display: block;`
- Each transition class (`.viz-transition-fade`, `.viz-transition-scale`, etc.) applies the animation on `.visible`

Example CSS pattern:
```css
.viz-panel.viz-transition-scale {
    transform: scale(0.85);
    transition: transform 0.6s cubic-bezier(0.16, 1, 0.3, 1), opacity 0.6s cubic-bezier(0.16, 1, 0.3, 1);
}
.viz-panel.viz-transition-scale.visible {
    transform: scale(1); opacity: 1;
}
.viz-panel.viz-transition-scale.exiting {
    transform: scale(0.85); opacity: 0;
    transition: transform 0.4s cubic-bezier(0.7, 0, 0.84, 0), opacity 0.4s cubic-bezier(0.7, 0, 0.84, 0);
}
```

For `reveal` and `wipe` which use clipPath, use CSS only (no keyframes needed):
```css
.viz-panel.viz-transition-reveal {
    clip-path: inset(100% 0 0 0);
    transition: clip-path 0.6s cubic-bezier(0.16, 1, 0.3, 1), opacity 0.6s cubic-bezier(0.16, 1, 0.3, 1);
}
.viz-panel.viz-transition-reveal.visible {
    clip-path: inset(0); opacity: 1;
}
```
