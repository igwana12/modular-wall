**campaign-page.md** — Full Kickstarter campaign page copy following the research-validated structure:
1. **Hook** (first paragraph): Spirit Sphere glowing, deity voice — the magical moment. Write for someone scrolling Kickstarter, 3 seconds to grab them.
2. **Problem** (1-2 paragraphs): "What if mythology could speak to you personally?" Frame as: ancient wisdom is locked in books, AI makes it conversational.

6. **Tiers and CTA** (reference tier-structure.md pricing, create urgency with early bird limits).
7. **AI transparency disclosure** — Kickstarter requires it. Readings powered by LLM (Claude), voices by ElevenLabs voice synthesis, visual art from PANTHEON pipeline. Be specific.
8. **Timeline** — Target Oct/Nov 2026 campaign, fulfillment May-June 2027. Show the timeline table from research.
9. **Open-source commitment** — Link to (future) GitHub repo. MIT firmware, CERN-OHL-S hardware, CC BY-SA enclosure.

**faq.md** — Backer FAQ covering at minimum:
- "What is a Maker Edition?" — Assembled, tested, open-source. Not a sealed consumer product. You can open it, modify it, flash custom firmware.
- "How does the AI work?" — Claude LLM for reading generation, ElevenLabs for deity voice synthesis, Pinecone RAG for mythology context. Requires WiFi.
- "Is it open source?" — Firmware MIT, hardware CERN-OHL-S, enclosure CC BY-SA. Deity voices and oracle prompts remain proprietary.
- "What deities are included?" — All 21 Greek gods from Sacred Circuits PANTHEON.
- "Does it work offline?" — Local LLM fallback via Ollama on home server. Requires setup. Default is cloud.
- "When will it ship?" — Target May-June 2027. Manufacturing timeline in campaign page.
- "International shipping?" — Via BackerKit post-campaign. Rates by region (from research: US $5-7 engine, $12-18 sphere; international varies).
- "Can I use my own AI?" — Yes — open-source firmware, swap the backend endpoint.
- "What's in the box?" — Oracle Engine: device + USB-C cable + card deck + quick-start guide. Spirit Sphere: everything in Engine + POV assembly + 3x 18650 batteries + assembly instructions.

**risks-and-challenges.md** — Kickstarter requires an honest risks section:
- First hardware project — mitigated by code-complete firmware, JLCPCB validated process, open-source community
- Manufacturing yield — budget for 15% loss, 10% contingency
- Shipping cost uncertainty — BackerKit post-campaign collection with real quotes
- Component supply chain — ESP32-S3 widely available, no exotic parts
- POV display fragility — Spirit Sphere ships with assembly instructions, not pre-assembled globe arm
- Timeline risk — holiday 2026 target, CNY-aware manufacturing schedule
- Tariff uncertainty — 10% BOM contingency for tariff changes

**stretch-goals.md** — Software/firmware ONLY (per research pitfall #1):
- $25K: 5 additional deity voices (total 26)
- $50K: Multi-language support (Spanish, French, Italian)
- $75K: Custom personality protocol creator (web UI)
- $100K: Ambient animation modes for Spirit Sphere (weather-reactive, music-reactive)
- $150K: Mobile companion app (reading history, deity favorites, voice settings)

Each stretch goal ships to ALL backers. No hardware additions. Explain why in the doc header.
