**Canvas Setup:**
- Canvas size: 3.0" x 5.0" (2.75" x 4.75" card + 0.125" bleed on all sides)
- At 300 DPI = 900 x 1500 pixels
- Install "Print for Figma" plugin from Figma Community for CMYK export and bleed marks

**Card Front Template (apply to all 21):**
Layout zones (see RESEARCH.md card layout diagram):
- BLEED zone: outer 0.125" on all sides (art extends here, will be trimmed)
- TRIM line: 2.75" x 4.75" actual card edge
- SAFE zone: 0.125" inside trim (keep all text/logos inside this)
- Content area: 2.5" x 4.5"

Content for each card front:
- PANTHEON deity art as primary visual (full bleed — art extends to canvas edge)
- Roman numeral number (I through XXI) — use mythological hierarchy ordering:
  I=Zeus, II=Hera, III=Poseidon, IV=Hades, V=Athena, VI=Apollo, VII=Artemis,
  VIII=Ares, IX=Aphrodite, X=Hermes, XI=Hephaestus, XII=Demeter,
  XIII=Dionysus, XIV=Persephone, XV=Hestia, XVI=Prometheus, XVII=Hecate,
  XVIII=Eros, XIX=Pan, XX=Nike, XXI=Tyche
- Deity name (from JSON config `name` field)
- Deity title (from JSON config `title` field)
- QR code: import SVG from assets/qr-codes/{deity_id}.svg, place at minimum 1.25" x 1.25"
- Use deity color_palette from JSON configs for accent colors

**Card Back (universal — 1 design):**
- Same 3.0" x 5.0" canvas with bleed
- Mystical/oracle themed pattern — consistent across all 21 cards
- Sacred Circuits branding element
- No text that identifies a specific deity

**Instruction Card (1 design):**
- Same 3.0" x 5.0" canvas
- Front side: "How to Use Your Oracle Cards"
  1. Choose a card that calls to you
  2. Scan the QR code with your phone camera
  3. Ask your question and receive your reading
  4. Each reading is unique — personalized by AI
- Back side: Sacred Circuits logo, oracleball.ai URL, brief brand story

**CMYK Warning:** PANTHEON art was created digitally (Midjourney) in RGB. When converting to CMYK:
- Blues and purples WILL lose saturation
- Use Print for Figma plugin to preview CMYK before export
- Consider boosting saturation 10-15% on the source art to compensate
- Order proof before bulk to verify color accuracy

**Export:**
- Use Print for Figma plugin to export each card as 300 DPI CMYK PDF with bleed marks
- Save to assets/card-designs/ directory:

  - assets/card-designs/back.pdf (1 file)
